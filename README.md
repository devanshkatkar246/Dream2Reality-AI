# Dream2Reality AI 🚀

Transform your dreams into structured career paths, interactive future simulations, and real-world Roadmaps.

## 🎯 Features
- **Dream Analysis**: AI-powered career path extraction, skill identification, and timeline estimation.
- **Future Simulation**: Immersive, choice-based future life experience simulator.
- **Reality Roadmap**: Actionable daily tasks, weekly goals, and projects to build.
- **Reality Score**: Feasibility score and honest feedback on your dream.
- **Premium UI**: Dark mode, glassmorphism, smooth animations (Framer Motion).

## ⚙️ Tech Stack
- **Frontend**: Next.js 14/15 (App Router, Tailwind CSS, Framer Motion)
- **AI Integration**: OpenAI (GPT-4o)
- **Icons**: Lucide React
- **Notifications**: React Toastify

## 🚀 Setup Instructions

1. **Clone & Install**:
   ```bash
   npm install
   ```

2. **Environment Setup**:
   - Create a `.env` file from `.env.example`.
   - Add your `OPENAI_API_KEY`.

3. **Run Locally**:
   ```bash
   npm run dev
   ```

4. **Experience the Dream**:
   - Open [http://localhost:3000](http://localhost:3000) in your browser.

## 🛠️ Folder Structure
- `app/`: Next.js pages and API routes.
- `components/`: Reusable UI components.
- `lib/`: AI helper functions and prompts.
- `public/`: Assets and static files.
